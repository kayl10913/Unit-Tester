## ğŸ“– How to Use

### 1. Code Editor Tab
- **Write or Paste Code**: Use the Monaco editor to write your code or paste existing code
- **Load Sample Code**: Click "Load Sample Code" to see the tool in action
- **Upload Files**: Upload existing code files (.js, .ts, .py, .java, .cpp, .c, .cs)
- **Language Support**: Switch between different programming languages
- **Code Statistics**: View real-time metrics about your code

### 2. Test Generator Tab
- **Select Framework**: Choose between Jest, Mocha, or Jasmine
- **Set Coverage Level**: Select the desired test coverage (Basic to Comprehensive)
- **Generate Tests**: Click "Generate Tests" to create AI-powered unit tests
- **Review Results**: Examine the generated tests in the editor
- **Download/Copy**: Save or copy the generated tests for use in your project

### 3. Vulnerability Checker Tab
- **Choose Scan Level**: Select from Basic, Standard, Comprehensive, or Deep scanning
- **Scan Code**: Click "Scan for Vulnerabilities" to analyze your code
- **Review Issues**: Examine detected vulnerabilities with severity levels
- **Security Score**: View your overall security score
- **Get Recommendations**: Read detailed remediation suggestions

### 4. Test Runner Tab
- **Select Environment**: Choose Node.js, Browser, or Jest environment
- **Run Tests**: Execute the generated tests against your code
- **View Results**: See detailed test execution results and coverage reports
- **Analyze Performance**: Review execution times and test statistics
